Классификация

По архитектуре

  • LAN (Local Area Network) — локальные сети, ограниченные зданием
  • WAN (Wide Area Network) — глобальные сети, охватывающие регионы или страны
  • MAN – Metropolian Area Network – Городские (территориальные) вычислительные сети

По масштабу администрирования

  • Сети рабочих групп: до 10–15 устройств, часто без выделенных серверов
  • Сети отделов: до 50 устройств с базовыми серверами (файловый, принт-сервер)
  • Корпоративные сети: сотни устройств, включая серверы баз данных, веб-серверы, системы облачного хранения

По уровню однородности

  • Однородные: все узлы используют одинаковые ОС и оборудование (например, сети на базе Windows Server)
  • Неоднородные: комбинация устройств с разными ОС (Windows, Linux, macOS) и протоколами

По способу подключения

  • Проводные: витая пара (Ethernet), оптоволокно, коаксиальный кабель
  • Беспроводные: Wi-Fi, Bluetooth, спутниковая связь, 5G
  • Гибридные: сочетание проводных и беспроводных технологий (например, офисная сеть с Wi-Fi и Ethernet)

По административным отношениям

  • Централизованные: управление через выделенные серверы
  • Децентрализованные (одноранговые): все узлы равноправны

По топологии

  • Шина: устройства подключены к общему кабелю
  • Звезда: все узлы соединены с центральным коммутатором
  • Кольцо: данные передаются по замкнутому контуру
